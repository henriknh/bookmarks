About

Hello! My name in Henrik Nilsson Harnert. And the reason I decided to create this website was because after browsing /r/unixporn for a while I noticed that many users had beautiful and personalized startup pages for their browsers.

So I wanted to create a easy tool to customize a startup page that everyone can use. More experienced user can modify their page with beautiful CSS and the less experienced user can choose an already made theme from the library!

This i a project I do in my spare time. If you have feedback or has found a bug don't hesitate to contact me!
